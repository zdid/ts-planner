🔄 Méthode 1 : Utiliser git pull dans VSCode (Recommandé)
VSCode a une intégration Git native qui permet de faire un git pull facilement.

Étapes pour ollama-sim et ts-planner
Ouvrez le terminal intégré dans VSCode :

Raccourci : Ctrl + \`` (ou Cmd + `` sur macOS).
Menu : Terminal → New Terminal.
Pour chaque projet (un par un) :

Sélectionnez le dossier du projet dans l'explorateur de fichiers (à gauche).
Dans le terminal, exécutez :
git pull origin main
Cela met à jour votre copie locale avec les dernières modifications poussées sur GitHub.
Si vous avez des conflits :

VSCode vous proposera de résoudre les conflits (généralement, vous pouvez accepter les modifications entrantes).
Sinon, exécutez :
git stash       # Sauvegarde vos modifications locales (si elles existent)
git pull origin main
git stash pop   # Restaure vos modifications locales (si nécessaire)
🔄 Méthode 2 : Utiliser l'Interface Git de VSCode
VSCode propose une interface graphique pour gérer Git.

Étapes
Ouvrez la barre latérale Git :

Ou utilisez le raccourci : Ctrl + Shift + G (ou Cmd + Shift + G sur macOS).
Pour chaque projet :

Sélectionnez le dépôt dans la liste déroulante (en haut de la barre Git).
Cliquez sur le bouton "Pull" (flèche vers le bas) pour récupérer les dernières modifications.
Si des modifications locales non commitées existent, VSCode vous proposera de les stasher (sauvegarder temporairement) avant de faire le pull.
Vérifiez les modifications :

Dans l'explorateur de fichiers, vous verrez les fichiers modifiés (marqués avec un "M" dans l'icône Git).
🔄 Méthode 3 : Vérifier la Branch et Faire un git fetch + git reset
Si vous n'êtes pas sûr d'être sur la bonne branch (main), voici comment forcer la synchronisation :

Ouvrez le terminal dans VSCode (`Ctrl + ``).
Pour chaque projet :
# Vérifiez la branch actuelle
git status

# Si vous n'êtes pas sur main, basculez :
git checkout main

# Récupérez les dernières modifications depuis GitHub
git fetch origin

# Réinitialisez votre copie locale pour qu'elle corresponde exactement à origin/main
git reset --hard origin/main
⚠️ Attention : git reset --hard supprime toutes vos modifications locales non commitées. Utilisez cette commande uniquement si vous n'avez pas de travail en cours à conserver.

class RulesLoader:
    """Charge les règles depuis MQTT (avec retain) ou le fichier local."""

    def __init__(self, path: str):
        self.path = Path(path)
        self._rules = ""
        self._lock = threading.Lock()
        self._load_initial()  # Charger initialement depuis MQTT ou fichier

    def _load_initial(self):
        """Charge les règles depuis MQTT (si disponible) ou le fichier local."""
        # Attendre que MQTT soit connecté et que les règles soient reçues via retain
        # (Le broker envoie automatiquement le dernier message retain après abonnement)
        # Si aucune règle n'est reçue après 2 secondes, charger depuis le fichier local
        time.sleep(2)  # Attendre que le message retain soit reçu

        with self._lock:
            if not self._rules and self.path.exists():
                with open(self.path, encoding="utf-8") as f:
                    content = f.read().strip()
                self._rules = content
                log.info(f"[rules] Règles chargées depuis {self.path} ({len(content)} caractères)")
            elif not self._rules:
                log.warning("[rules] Aucune règle disponible (MQTT et fichier local introuvables)")

    def get(self) -> str:
        with self._lock:
            return self._rules

    def update_from_mqtt(self, rules: str):
        """Met à jour les règles depuis MQTT."""
        with self._lock:
            self._rules = rules
        log.info(f"[rules] Règles mises à jour via MQTT ({len(rules)} caractères)")

rules_loader = RulesLoader(RULES_FILE)

# Dictionnaire des futures en attente : correlation_id → asyncio.Future
_pending: dict[str, asyncio.Future] = {}
_pending_lock = threading.Lock()
_loop: asyncio.AbstractEventLoop | None = None

def _on_mqtt_message(client, userdata, message):
    """Appelé dans le thread MQTT — résout la Future correspondante."""
    try:
        payload = json.loads(message.payload.decode())
        corr_id = payload.get("correlation_id")
        if not corr_id:
            return
        with _pending_lock:
            future = _pending.get(corr_id)
        if future and _loop:
            _loop.call_soon_threadsafe(future.set_result, payload)
    except Exception as e:
        log.error(f"[mqtt] Erreur réception : {e}")

async def mqtt_send_and_wait(payload: dict) -> dict:
    """
    Publie sur TOPIC_COMMANDE et attend la réponse sur TOPIC_REPONSE.
    Utilise un correlation_id pour matcher la bonne réponse.
    Timeout : MQTT_TIMEOUT_SEC secondes (2s).
    """
    if not TS_PLANNER_ONLINE:
        log.warning("[mqtt] ts-planner est OFFLINE, passage en mode dégradé")
        return {"error": "offline", "message": "ts-planner est hors ligne"}

    corr_id = str(uuid.uuid4())
    payload["correlation_id"] = corr_id

    future = asyncio.get_event_loop().create_future()
    with _pending_lock:
        _pending[corr_id] = future

    try:
        mqtt_client.publish(TOPIC_COMMANDE, json.dumps(payload, ensure_ascii=False), qos=1)
        log.info(f"[mqtt] Publié sur {TOPIC_COMMANDE} (corr={corr_id[:8]}...)")
    except Exception as e:
        log.error(f"[mqtt] Échec publication MQTT: {e}. Mode dégradé activé.")
        return {"error": "mqtt_error", "message": f"Erreur MQTT: {e}"}

    try:
        result = await asyncio.wait_for(future, timeout=MQTT_TIMEOUT_SEC)
        log.info(f"[mqtt] Réponse reçue (corr={corr_id[:8]}...)")
        return result
    except asyncio.TimeoutError:
        log.error(f"[mqtt] Timeout après {MQTT_TIMEOUT_SEC}s (corr={corr_id[:8]}...)")
        return {"error": "timeout", "message": "Le programme de planification ne répond pas"}
    finally:
        with _pending_lock:
            _pending.pop(corr_id, None)

def build_messages(body: dict) -> list[dict]:
    messages = list(body.get("messages", []))
    if not messages:
        system = body.get("system", "")
        prompt = body.get("prompt", "")
        if system:
            messages.append({"role": "system", "content": system})
        if prompt:
            messages.append({"role": "user", "content": prompt})
    else:
        system = body.get("system", "")
        if system and not any(m.get("role") == "system" for m in messages):
            messages = [{"role": "system", "content": system}] + messages
    return messages

def inject_rules(messages: list[dict]) -> list[dict]:
    """
    Injecte les règles domotiques à la fin du system prompt existant.
    Si pas de system prompt, en crée un.
    """
    rules = rules_loader.get()
    if not rules:
        return messages

    messages = list(messages)
    for i, m in enumerate(messages):
        if m.get("role") == "system":
            messages[i] = {
                **m,
                "content": m["content"] + "\n\n" + rules
            }
            return messages

    # Pas de system prompt → en créer un
    messages.insert(0, {"role": "system", "content": rules})
    return messages

def extract_json_from_text(text: str) -> dict | None:
    """
    Tente d'extraire un JSON structuré de la réponse texte de Mistral.
    Retourne le dict si "type" est un type connu, sinon None.
    """
    text = text.strip()
    # Nettoyer les balises markdown
    if text.startswith("```"):
        lines = text.split("\n")
        text  = "\n".join(lines[1:-1] if lines[-1] == "```" else lines[1:])
    try:
        data = json.loads(text)
        if isinstance(data, dict) and data.get("type") in STRUCTURED_TYPES:
            return data
    except Exception:
        pass
    return None

# ─── Streaming Mistral → Ollama NDJSON ────────────────────────────────────────

STRUCTURED_TYPES = {"planification", "macro", "macro_ref", "condition",
                    "sequence", "gestion", "execution"}

async def stream_mistral_to_ollama(
    mistral_stream: httpx.Response,
    ollama_model:   str,
) -> AsyncIterator[str]:
    prompt_tokens     = 0
    completion_tokens = 0
    text_parts:       list[str]  = []
    tool_calls:       list[dict] = []
    tool_call_buffer: dict[int, dict] = {}

    async for line in mistral_stream.aiter_lines():
        if not line or not line.startswith("data: "):
            continue
        payload = line[6:].strip()
        if payload == "[DONE]":
            break
        try:
            chunk = json.loads(payload)
        except json.JSONDecodeError:
            continue

        choice        = chunk.get("choices", [{}])[0]
        delta         = choice.get("delta", {})
        content       = delta.get("content") or ""
        finish_reason = choice.get("finish_reason")

        for tc_delta in delta.get("tool_calls", []):
            idx  = tc_delta.get("index", 0)
            func = tc_delta.get("function", {})
            if idx not in tool_call_buffer:
                tool_call_buffer[idx] = {
                    "id":       tc_delta.get("id", ""),
                    "type":     tc_delta.get("type", "function"),
                    "function": {"name": "", "arguments": ""},
                }
            if func.get("name"):
                tool_call_buffer[idx]["function"]["name"]      += func["name"]
            if func.get("arguments"):
                tool_call_buffer[idx]["function"]["arguments"] += func["arguments"]

        usage = chunk.get("usage") or {}
        if usage:
            prompt_tokens     = usage.get("prompt_tokens",     prompt_tokens)
            completion_tokens = usage.get("completion_tokens", completion_tokens)

        if finish_reason in ("stop", "tool_calls"):
            if content:
                text_parts.append(content)
                yield json.dumps({
                    "model":      ollama_model,
                    "created_at": now_iso(),
                    "message":    {"role": "assistant", "content": content},
                    "done":       False,
                }, ensure_ascii=False) + "\n"
            break

        if content:
            text_parts.append(content)
            yield json.dumps({
                "model":      ollama_model,
                "created_at": now_iso(),
                "message":    {"role": "assistant", "content": content},
                "done":       False,
            }, ensure_ascii=False) + "\n"

    for tc in tool_call_buffer.values():
        try:
            tc["function"]["arguments"] = json.loads(tc["function"]["arguments"])
        except Exception:
            pass
        tool_calls.append(tc)

    done_chunk = make_ollama_done_chunk(ollama_model, prompt_tokens, completion_tokens)
    if tool_calls:
        done_chunk["message"]["tool_calls"] = tool_calls

    yield json.dumps(done_chunk, ensure_ascii=False) + "\n"
    yield json.dumps({"__meta__": True, "text": text_parts, "tool_calls": tool_calls}) + "\n"
